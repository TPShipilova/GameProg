# Программирование игр

Шипилова Татьяна, М8О-406Б-22

## Механики

1) Конструктор ракет
(выбор модулей,
имеющих разные характеристики,
влияющие на полет)
2) Симуляция физики полета
(в т.ч. закона всемирного тяготения,
разные подвижные
космические тела (планеты))
+ визуальные эффекты Particle System


## Реализация

Выбор частей ракеты: ScriptableObjects + статический класс + префабы.

Управление ракетой: UnityEngine.Input (legacy), физика тяги через AddForceAtPosition.

Две планеты и ракета подчиняются закону Всемирного тяготения (RigidBody + код + убрано дефолтное ускорение свободного падения).

Эффект пламени ракеты с помощью ParticleSystem.

UI через Canvas.
